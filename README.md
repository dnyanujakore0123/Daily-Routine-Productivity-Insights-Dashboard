# Daily-Routine-Productivity-Insights-Dashboard
This project analyzes daily habits like screen time, exercise, wake-up time, and meditation to understand their impact on productivity levels.

🔹 Tools Used

Power BI

Power Query

DAX

🔹 Key Insights

Average productivity score: 7.0

Higher screen time leads to lower productivity

People who exercise have better productivity scores

Meditation and structured routines improve consistency

Average wake-up time around 7 AM

🔹 Features

Scatter plot for screen time vs productivity

Exercise-based productivity comparison

KPI cards (Productivity, Screen Time, Wake-up Time)

Interactive filters (Exercise, Meditation)
